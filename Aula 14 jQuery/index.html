React

É uma biblioteca JavaScript de código aberto para a criação de interfaces de usuário. Foi desenvolvida pelo Facebook e permite a criação de componentes de UI reutilizáveis, que gerenciam seu próprio estado.

Angular

É uma plataforma de aplicações web de código-fonte aberto, baseado em TypeScript e liderado pela equipe do Google. É um framework completo que oferece uma estrutura robusta para o desenvolvimento de aplicações de página única (SPAs).

Vue.js

É um framework progressivo para a construção de interfaces de usuário. Diferente de frameworks monolíticos, o Vue foi projetado para ser adotado de forma incremental, focando apenas na camada de visualização.

Ember.js

É um framework JavaScript produtivo e testado em batalha para a construção de aplicações web modernas. Ele inclui tudo o que é necessário para construir interfaces de usuário ricas que funcionam em qualquer dispositivo.

Preact

É uma alternativa rápida e leve ao React, com apenas 3kB de tamanho. Ele oferece a mesma API moderna do React e tem como objetivo fornecer uma alternativa de baixo custo em termos de tamanho de arquivo, sem sacrificar a performance.
